یه دستور minikube ip بزن
توی مرورگر برو توی این آدرس: http://<Minikube-IP>:30880
user: root
password: همونی که ست کردی خودت

یه پروژه جدید بساز از پایین تیک readme رو بردار الکی readme نسازه
اسم پروژه: cloud-project

وقتی پروژه ساختی برو به: Settings -> CI/CD -> Runner

رانر بساز اسمش هم این باشه: k8s-runner
واسه ساخت رانر عکس تنظیماتشو توی تلگرام میدم

همون موقع که رانر ساخته میشه یه توکن بهت میده فقط هم همین یک بار. پس ذخیرش کن

بعد فایل runner-values رو طبق عکسی که توی تلگرام بهت میدم درست کن براساس اطلاعات خودت

حالا این دستورات رو توی ترمینال بزن
helm repo add gitlab https://charts.gitlab.io
helm repo update
helm install gitlab-runner gitlab/gitlab-runner -n cc-project -f gitlab/runner-values.yaml
kubectl create rolebinding default-admin --clusterrole=admin --serviceaccount=cc-project:default -n cc-project

# ارسال به گیت لب
git remote add gitlab http://<Minikube-IP>:30880/root/cloud-project.git
git push -u gitlab main (اسم برنچت ممکنه فرق داشته باشه. اسم برنج خودتو وارد کن)
