kubectl apply -f k8s/04-traffic-generator.yaml

after running last command, use below command in another terminal to see log live:
kubectl exec -it deployment/nginx-gateway -n cc-project -- tail -f /var/log/nginx/nginx_access.log

after finish generating all logs, use below command to run map reduce:
kubectl apply -f k8s/05-mapreduce.yaml

while it is running, in other terminal use this command to see bash log live:
kubectl logs -f job.batch/mapreduce-pipeline -n cc-project

and after bash finish its job, use this last command to see final summary.json:
kubectl exec -it deployment/nginx-gateway -n cc-project -- cat /var/log/nginx/outputs/final/summary.json
